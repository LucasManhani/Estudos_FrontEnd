# 📁 Estudos Front-End

Repositório com projetos desenvolvidos durante meu período de aprendizado em desenvolvimento Front-End. Atualmente estou em transição para **Engenharia de Dados**.

> ⚠️ Estes projetos não estão mais sendo mantidos e representam uma fase anterior de aprendizado.

---

## 📌 Projetos

### 🎮 Kingdom Hearts Site
Página de blog temática inspirada no jogo Kingdom Hearts.

**Ferramentas e conceitos utilizados:**
- HTML5 semântico
- CSS3 (estilização, layout e responsividade)
- Organização de assets (imagens e áudio)

---

### 🛒 Form de Vendas
Formulário de vendas com campos estruturados para coleta de informações.

**Ferramentas e conceitos utilizados:**
- HTML5 (formulários, inputs e validação nativa)
- CSS3 (estilização de formulários)

---

### 📱 Gerador de QR Code
Aplicação para geração de QR Codes a partir de texto ou URL.

**Ferramentas e conceitos utilizados:**
- HTML5 e CSS3
- JavaScript
- Integração com biblioteca externa de geração de QR Code

---

### ✅ To Do List
Lista de tarefas com funcionalidades de busca, filtro e persistência de dados.

**Ferramentas e conceitos utilizados:**
- HTML5 e CSS3
- JavaScript (manipulação do DOM)
- LocalStorage (persistência de dados no navegador)
- Filtro e busca dinâmica de tarefas
